# 챕터1: 가짜뉴스 판단 능력 기르기
#챕터1-소피아박사
플레이어가 연구소에 도착하자마자 감염된 AI들이 생성한 여러 가짜뉴스가 동시에 쏟아지는 상황. 혼란 속에서 숨어있던 소피아 베리타스 박사를 찾아 구출하고, 첫 번째 디지털 윤리 훈련을 받게 된다.

> **현재 확산 중인 가짜뉴스들**

- "제주도 신종 바이러스 발견, 치사율 78%"
- "OOO사 2,500만명의 디지털 디바이스 정보 유출"
- "마스크 착용이 오히려 감염 위험을 높인다"

| 등장인물   | 대화                                                                                                                                               |
| ------ | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| 소피아 박사 | 어서오세요, 당신이 마지막 가디언이군요. 소식을 많이 들었지만 이렇게 가까이 뵙는 것은 처음이네요. 소피아입니다.                                                                                  |
| 플레이어   | **[선택지 1]** "박사님, 안전합니다. 상황이 어떻게 되고 있나요?"                                                                                                        |
| 소피아 박사 | 슈도스 바이러스가 우리가 예상했던 것보다 훨씬 더 진화했습니다. <br>단순히 거짓 정보를 만드는 것을 넘어서, 사람들의 인지 편향을 악용해 진실과 거짓을 구별할 수 없게 만들고 있어요.                                         |
|        | 방금 전에도 제주도 바이러스 관련 가짜뉴스가 퍼졌는데, 벌써 수천 명이 믿고 있어요. '코로나보다 심각한 바이러스 발견, 치사율 78%'라는 헤드라인이었는데...                                                       |
| 플레이어   | 그 뉴스라면 저도 봤습니다. 어떻게 가짜라고 확신할 수 있나요?                                                                                                              |
| 소피아 박사 | 좋은 질문이에요! 바로 그런 의구심이 필요한 거죠. [태블릿을 보여주며] 보세요, 이 태블릿에 실제 WHO 데이터가 있어요. 제주도 보건소 공식 발표도 있고요. 하지만 더 중요한 건, 이런 검증 과정을 스스로 할 수 있는 능력을 기르는 것입니다.        |
|        | 당신의 알레테이아 칩에는 디지털 백신의 핵심 정보가 들어있어요. 하지만 백신만으로는 부족해요. 사람들이 스스로 생각하는 능력을 되찾아야 합니다.                                                                 |
|        | 첫 번째 임무를 드릴게요. 이 연구소에 남아있는 세 개의 터미널에서 서로 다른 가짜뉴스들을 분석해보세요. 각각 다른 유형의 조작 기법을 사용하고 있어요.                                                            |
| 플레이어   | 어떤 기준으로 판별해야 하나요?                                                                                                                                |
| 소피아 박사 | 기억하세요 - **출처, 신뢰성, 직감**. 이 세 가지가 핵심입니다.                                                                                                          |
|        | 구체적으로 설명드릴게요:  <br>  <br>**출처 확인**: 누가, 언제, 어디서 발표했는가?  <br>- 원본 기사의 작성자와 언론사를 확인하세요  <br>- 발행 날짜와 시간을 체크하세요<br>- 1차 정보원까지 추적해보세요                |
|        | **신뢰성 검증**: 정보 제공자를 믿을 만한가?  <br>- 해당 언론사의 평판과 전문성을 확인하세요  <br>- 작성자가 해당 분야 전문가인지 파악하세요  <br>- 편향성이나 특정 목적이 있는지 살펴보세요                            |
|        | **직감적 판단**: 너무 자극적이거나 비현실적이지 않은가?  <br>- 감정을 과도하게 자극하는 표현은 의심해보세요  <br>- '긴급', '충격', '절대 비밀' 같은 자극적 단어들을 경계하세요  <br>- 상식적으로 납득하기 어려운 내용인지 생각해보세요 |
| 플레이어   | 만약 확신이 서지 않으면 어떻게 하나요?                                                                                                                           |
| 소피아 박사 | **출처, 신뢰성, 직감**을 기억하세요. 그리고 **교차검증**을 통해 얻은 정보로 냉정한 판단을 내리는 것이야말로 우리의 가장 강력한 무기입니다.                                                              |
|        | 확실하지 않을 때는 성급하게 공유하지 마세요. **'모른다'고 인정하는 것**도 용기입니다.                                                                                              |
|        | 완벽한 알레테이아 칩을 만들기 위해서는 **출처는 진실의 나침반, 신뢰는 신뢰도 측정기, 직감은 감정 필터**가 필요합니다.<br>행운을 빕니다. 가디언.                                                           |
## 목표
| 템      | 별명                                       | 설명                                                                            |
| ------ | ---------------------------------------- | ----------------------------------------------------------------------------- |
| **출처** | **"진실의 나침반"  <br>(Truth Compass)**       | 네메스에서 제작한 진실의 나침반.<br>누가, 언제, 어디서 작성했는지 추적할 수 있다. <br>거짓 정보일수록 바늘이 불안정하게 떨린다. |
